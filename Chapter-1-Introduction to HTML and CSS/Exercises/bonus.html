Common HTML Tags

Tag Description Example
<h1> Main heading <h1>Title</h1>
<p> Paragraph <p>Text here</p>
<a> Link <a href="#">Click me</a>
<img> Image <img src="img.jpg" alt="desc">
<ul> Unordered list <ul><li>Item</li></ul>
<ol> Ordered list <ol><li>First</li></ol>
<form> Form container <form><input></form>
<input> Input field <input type="text">
<button> Button <button>Click</button>
<table> Table <table><tr><td>Data</td></tr></table>

Essential CSS Properties

Property Description Example
color Text color color: red;
background Background styling background: #f0f0f0;
margin Outer spacing margin: 10px;
padding Inner spacing padding: 15px;
border Element border border: 2px solid black;
font-size Font size font-size: 18px;
text-align Text alignment text-align: center;
box-shadow Add shadow to elements box-shadow: 2px 2px 5px rgba(0,0,0,0.3);
width Width of the element width: 100%;
height Height of the element height: 50px;
