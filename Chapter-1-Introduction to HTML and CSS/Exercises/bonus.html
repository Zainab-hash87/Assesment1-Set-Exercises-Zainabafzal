HTML & CSS Quick Reference Guide

A quick reference for common HTML tags and CSS properties.

HTML Tags

<h1> to <h6>

Defines HTML headings. <h1> is the highest level, <h6> is the lowest.

<h1>This is a heading</h1>

<p>

Defines a paragraph.

<p>This is a paragraph.</p>

<a>

Defines a hyperlink.

<a href="https://example.com">This is a link</a>

<ul> and <li>

Defines an unordered list and list items.

<ul> 
    <li>Item 1</li> 
    <li>Item 2</li> 
</ul>

<ol> and <li>

Defines an ordered list and list items.

<ol> 
    <li>First item</li> 
    <li>Second item</li> 
</ol>

<img>

Defines an image.

<img src="image.jpg" alt="Description">

<div>

Defines a division or section in an HTML document.

<div>Content here</div>

<span>

Defines a section in a document, typically used for styling.

<span style="color: red;">Red text</span>

<form>

Defines an HTML form for user input.

<form> 
    <input type="text" placeholder="Enter text"> 
    <input type="submit" value="Submit"> 
</form>

<table>

Defines a table.

<table> 
    <tr> 
        <th>Header</th> 
        <th>Header</th> 
    </tr>
    <tr> 
        <td>Data</td>
